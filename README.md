 Personal Finance Analytics Dashboard

## Objective

The Personal Finance Analytics Dashboard is designed to help individuals track and analyze their financial activities. The dashboard provides insights into income, expenses, savings, and spending behavior, enabling users to make informed financial decisions and improve budgeting practices.

## Tools Used

* Power BI (Dashboard Development & Visualization)
* Microsoft Excel (Data Cleaning & Preparation)
* SQL (Data Analysis & Querying)

## Features

* Interactive financial dashboard
* Income and expense tracking
* Savings monitoring
* Category-wise spending analysis
* Monthly financial performance overview
* Dynamic filters for customized analysis
* KPI cards for quick financial insights

## Income Analysis

* Total income earned
* Income trends over time
* Income contribution by source
* Monthly income comparison
* Highest income-generating categories

## Expense Analysis

* Total expenses incurred
* Expense breakdown by category
* Monthly expense trends
* Top spending categories
* Expense-to-income ratio analysis

## Savings Tracking

* Total savings accumulated
* Monthly savings trends
* Savings rate percentage
* Comparison between income, expenses, and savings
* Financial goal tracking and performance monitoring
